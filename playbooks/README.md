# Playbooks por área

Cómo se trabaja día a día en cada disciplina. Todos los playbooks comparten la misma estructura: el cambio de paradigma, el ciclo de trabajo, las reglas no negociables y la métrica que optimizan (definida en [metricas.md](../modelo-operativo/metricas.md)).

| Playbook | Paradigma |
|---|---|
| [ingenieria.md](ingenieria.md) | Ingeniería compuesta: escribir código es barato; diseñar y revisar es el trabajo |
| [producto.md](producto.md) | PM agent-native: estrategia como contexto ejecutable, monitoreo conversacional |
| [diseno-ux.md](diseno-ux.md) | Interfaces de intención y sistemas de diseño computables |
| [marketing.md](marketing.md) | Gráficos de contexto y atomización gobernada de contenido |
